PointCloudViewer

new PointCloudViewer(targetElement, params)

Source:
  • point-cloud-viewer.js, line 75

Setup a point cloud viewer on the given target element.

Parameters:
Name Type Description
targetElement HTMLDivElement

An element under which the renderer will be mounted.
params Object

Initialization parameters.

Properties
Name Type Attributes Description
server String

The server address where the greyhound point cloud resource is located.
resource String

The name of the greyhound resource.
brushes Array.<String>

An array of brush specs.
allowGreyhoundCredentials Boolean <optional>

Whether to send a credentials flag. Defaults to false.
rendererOptions Object <optional>

An arbitrary object to setup initial renderer options.
initialCameraParams Object <optional>

Serialized camera properties to setup the initial camera position.
filter Object <optional>

Initial filter for querying entwine data.
baseDepth Number <optional>

The base depth for the resource. Defaults to 8.
cameraChangeCallbackFn cameraChangeCallbackFn <optional>

A callback function to call when camera changes. This is different from a "view-changed" notification. This function is called for all camera changes (e.g. through animated transitions etc.).

setColorChannelBrushes(brushSpecs)

Source:
  • point-cloud-viewer.js, line 286

Set the current set of channel brushes. Upto 4 brushes may be specified. The brush channels need to be specified in order which correspond to the 4 color channels. E.g. if the first brush needs to be unset, the brush spec at index 0 needs to be null.

Example
pointCloudViewer.setColorChannelBrushes([null, 'local://color']);
Parameters:
Name Type Description
brushSpecs Array.<String>

An array of brush specs with no more than 4 elements.
