subdivide() → {Array.<Box>}

Source:
  • frustum-lod.js, line 315

Subdivide a box into children boxes. Parameters stopSplitDepth and hardStopDepth control how splitting happens. While the box's depth is lower than stopSplitDepth, each box is split into 8 children. After that, the box is not split, but instead is "subdivided" into a single child box with the same bounds as this box but with depth ranging from current box's depth to hardStopDepth.

Returns:

An array of child boxes, could either be 8, 1 or 0 in count.
