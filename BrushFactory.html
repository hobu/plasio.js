BrushFactory

BrushFactory

Creates brushes based on a URL spec, allows for registration and de-registration of brush types. The library provides several stock brushes which may be queried using BrushFactory.availableBrushes function.

Custom brushes derived from the BaseBrush class may be registered which then become available for use. The BrushFactory.registerBrush may be used to do so.

class MapboxBrush extends BaseBrush {
    // ...
}

// Makes the brush available as tms://mapbox?...
BrushFactory.registerBrush('tms', 'mapbox', MapboxBrush);
